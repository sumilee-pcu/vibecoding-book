# 표 6-1. CLAUDE.md의 여섯 항목

CLAUDE.md에는 프로젝트를 처음 맡은 사람이 알아야 할 정보를 여섯 항목으로 적습니다. 각 항목에 넣을 내용과 짧은 예시를 정리합니다.

| 항목 | 넣을 내용 | 짧은 예시 |
| --- | --- | --- |
| 프로젝트 목적 | 무엇을 하는 프로젝트인가 | 할 일을 추가·완료하는 Flask 웹앱 |
| 실행 명령 | 어떻게 실행하나 | `flask --app app run` |
| 테스트 명령 | 어떻게 검증하나 | `pytest -q` |
| 코드 규칙 | 지켜야 할 약속 | 새 라우트는 테스트 먼저, 함수명은 동사로 |
| 완료 기준 | 어디까지 되면 끝인가 | 테스트 통과·화면 확인·diff 검토 |
| 보고 방식 | 결과를 어떻게 알리나 | 변경 파일·실행 명령·남은 위험 |

노트: 여섯 항목은 한 번 적어 두면 매 프롬프트에서 반복하지 않아도 AI가 항상 참고합니다.
