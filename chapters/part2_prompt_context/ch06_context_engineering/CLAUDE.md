# CLAUDE.md

이 파일은 AI가 이 프로젝트에서 일할 때 항상 참고하는 규칙입니다. 아래 예시값은 Flask 할 일 관리 앱을 기준으로 채워 두었으니, 자기 프로젝트에 맞게 바꿔 쓰세요.

## 프로젝트 목적
- 할 일을 추가하고 목록을 보며 완료를 토글하는 Flask 웹앱입니다.
- 학습용 예제이므로 단순함을 우선합니다.

## 실행 명령
- 개발 서버 실행: `flask --app app run`
- 접속 주소: http://127.0.0.1:5000

## 테스트 명령
- 전체 테스트: `pytest -q`

## 코드 규칙
- 새 라우트를 추가할 때는 테스트를 먼저 작성합니다(테스트 우선).
- 외부 라이브러리는 함부로 추가하지 않습니다.
- 함수 이름은 동사로 시작합니다.

## 완료 기준
- ① 테스트가 모두 통과합니다(`pytest -q`).
- ② 화면에서 추가·완료 토글이 실제로 동작합니다.
- ③ 변경 내용(diff)을 검토했습니다.

## 보고 방식
- 변경한 파일 목록을 알려 줍니다.
- 실행에 필요한 명령을 적어 줍니다.
