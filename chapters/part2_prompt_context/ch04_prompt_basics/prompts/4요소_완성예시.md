# 네 요소를 모두 갖춘 프롬프트 예시

역할·맥락·지시·형식이 한 문단 안에 모두 들어간 예시입니다. 대괄호 표기는 각 문장이 어떤 요소를 맡는지 보여 줍니다.

## 완성 예시
"결제 모듈을 검토하는 보안 리뷰어로서 [역할], 방금 추가한 payment.js의 카드 정보 처리 코드를 [맥락], 민감 정보가 로그나 응답에 노출되는지 중심으로 점검해 줘 [지시]. 위험이 높은 순서로, 각 항목마다 위치와 고치는 방법을 표로 정리해 줘 [형식]."

## 요소별 해설
- 역할: "보안 리뷰어로서" — 어떤 관점으로 볼지 좁혀 줍니다.
- 맥락: "방금 추가한 payment.js의 카드 정보 처리 코드" — 대상 파일과 상황을 줍니다.
- 지시: "민감 정보가 로그나 응답에 노출되는지 중심으로 점검" — 무엇을 할지 구체화합니다.
- 형식: "위험이 높은 순서로 … 표로 정리" — 결과의 모양을 지정합니다.

노트: 네 요소 중 특히 맥락과 형식이 자주 빠집니다. 이 둘을 의식적으로 채우면 결과가 안정됩니다.
