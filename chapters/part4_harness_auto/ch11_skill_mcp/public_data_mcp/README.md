# 공공데이터 MCP

공공데이터 MCP는 법령·통계·날씨 같은 공공 데이터를 근거로 끌어오는 MCP입니다. 모델이 기억에만 의존하면 사실과 다른 답(환각, hallucination)을 낼 수 있습니다. 공공데이터 MCP는 답의 근거를 실제 출처에서 가져와 환각을 줄입니다.

## 왜 쓰는가

- 법령 조항, 통계 수치, 날씨 정보처럼 정확성이 중요한 답에 근거를 붙입니다.
- 모델이 가져온 원자료와 모델이 요약한 내용을 구분해 검증할 수 있습니다.
- 출처 URL이 남으므로 사람이 원문을 확인할 수 있습니다.

## 어떻게 시작하는가

키 없이 시작할 수 있는 예제 서버로 입문하길 권합니다. 처음부터 인증이 필요한 서버를 붙이면 연결 문제가 인증 문제와 섞여 디버깅이 어렵습니다. 동작을 익힌 뒤 키가 필요한 서버로 넘어갑니다.

이 폴더에는 두 가지 예가 있습니다.

- `law-kr/`: 법령 검색 MCP에 첫 질의를 던지는 예.
- `book-data/`: ISBN으로 도서 정보를 조회하는 교육용 MCP 서버 스텁.

## 한계

공공데이터 MCP는 근거를 가져오는 장치이지 전문가 판단을 대신하지 않습니다. 법령을 조회했다고 해서 그 답이 법률 자문이 되는 것은 아닙니다. 가져온 자료는 원문 확인이 필요한 참고 정보로 다루고, 판단은 사람이 합니다.
