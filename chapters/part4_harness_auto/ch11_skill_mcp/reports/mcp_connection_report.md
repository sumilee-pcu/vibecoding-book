## 작성 예시: book-data

- 연결 목적: ISBN으로 도서 정보를 조회해 메모 작성에 참고
- 서버 이름: book-data
- 연결 방식: stdio(로컬 server.py)
- scope: user
- 인증: 없음(샘플 스텁). 실제 네이버 provider 사용 시 환경변수로 토큰 주입
- 읽기 권한: 있음(lookup_isbn 조회)
- 쓰기 권한: 없음
- 첫 질의: ISBN 9780134685991의 도서 정보를 출처와 함께 조회
- 사용한 도구: lookup_isbn
- 반환된 원자료: 제목 Effective Java / 저자 Joshua Bloch / 출처 openlibrary.org
- 모델 요약: 자바 모범 사례를 다루는 책으로, 도서 메모의 참고 도서로 적합
- 원자료와 요약의 차이: 제목·저자·출처는 원자료. "참고 도서로 적합"은 모델 판단(근거 외 의견)
- 보안 점검: 읽기 전용, 토큰 없음, 외부 전송 없음. 위험 행동 없음
- 유지·제거 결정: 유지. 읽기 전용이고 조회 결과가 출처와 함께 검증 가능
