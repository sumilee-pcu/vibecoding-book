# 작업 규칙(CLAUDE.md)

도서 메모 검색 도구(filterNotes) 프로젝트의 작업 규칙입니다. 모든 작업은 아래 규칙을 따릅니다.

1. 작업 전 SPEC.md를 먼저 읽습니다.
2. 수정 전 계획을 제시하고 승인을 받습니다.
3. 한 번에 하나의 기능 또는 버그만 다룹니다.
4. 완료 기준과 연결되지 않는 코드는 추가하지 않습니다.
5. 테스트를 느슨하게 바꾸거나 삭제하지 않습니다.
6. 구현 뒤에는 테스트를 실행하고 결과를 요약합니다.
7. 파일 삭제·외부 전송·데이터 수정·배포는 승인 없이 하지 않습니다.
8. 커밋 전 diff에서 범위 침범이 없는지 확인합니다.
