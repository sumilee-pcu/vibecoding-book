# 명세: 도서 메모 검색 도구(filterNotes)

## 프로젝트 이름

도서 메모 검색 도구(filterNotes)

## 목적

사용자가 저장한 도서 메모를 제목·저자·메모 내용으로 빠르게 검색합니다.

## 이번 범위

- 제목으로 검색
- 저자로 검색
- 메모 내용으로 검색
- 빈 검색어일 때 전체 표시
- 결과가 없을 때 표시

## 제외 범위

- 저장 방식 변경(이번 작업에서는 손대지 않습니다)
- 메모 추가·수정·삭제 기능
- 정렬·필터 고도화

## 주요 데이터

도서 메모 한 건은 제목, 저자, 메모 내용을 가집니다. 검색은 이 세 필드를 대상으로 합니다.

## 제약

- 검색은 원본 데이터를 바꾸지 않습니다.
- 대소문자를 구분하지 않습니다.
- 검색어 앞뒤 공백은 무시합니다.
- 저장 방식은 이번 작업에서 바꾸지 않습니다.

## 완료 기준

- 제목으로 검색이 동작합니다.
- 저자로 검색이 동작합니다.
- 메모 내용으로 검색이 동작합니다.
- 빈 검색어를 넣으면 전체가 표시됩니다.
- 일치 결과가 없으면 결과 없음이 표시됩니다.
- 검색 전후로 원본 데이터가 바뀌지 않습니다.

## 자동 확인

- 검색 함수(filterNotes) 단위 테스트로 위 완료 기준을 확인합니다.
- 화면 체크리스트 5개:
  - ☐ 제목 일부로 검색하면 해당 메모가 나온다.
  - ☐ 저자 일부로 검색하면 해당 메모가 나온다.
  - ☐ 메모 내용 일부로 검색하면 해당 메모가 나온다.
  - ☐ 빈 검색어면 전체가 나온다.
  - ☐ 없는 단어로 검색하면 결과 없음이 나온다.

## 수동 확인

- 대소문자를 섞어 검색해도 같은 결과가 나오는지 눈으로 확인합니다.
- 앞뒤 공백을 넣어도 같은 결과가 나오는지 눈으로 확인합니다.

## 남은 위험

- 메모 양이 많아질 때의 검색 속도는 이번 범위에서 다루지 않습니다.
- 한글·영문 혼용 검색의 세부 동작은 추후 확인이 필요합니다.
