## 기록 예시

- 날짜: 2026-06-20
- 작업: 메모 내용 검색에서 대소문자 구분 문제 수정
- 연결된 완료 기준: 메모 내용으로 검색이 동작한다 / 대소문자를 구분하지 않는다
- 사용한 프롬프트: SPEC.md와 CLAUDE.md를 먼저 읽고, filterNotes 함수에서 메모 내용 검색이 대소문자를 구분하는 문제를 계획부터 제시해 줘. 승인 전에는 코드를 바꾸지 마.
- 수정 파일: src/filterNotes.js, tests/filterNotes.test.js
- 실행한 테스트: 단위 테스트 6건 실행, 처음 1건 실패 후 전체 통과
- 실패와 수정: "MEMO" 대문자 검색이 메모 내용에서만 결과를 놓침. 검색어와 메모 내용을 모두 소문자로 변환하도록 한 줄 수정.
- diff 검토 결과: 변경은 filterNotes 함수 내부 한 곳과 테스트 1건 추가뿐. 범위 침범 없음.
- 커밋: fix: 메모 내용 검색을 대소문자 무시로 통일
- 남은 위험: 한글·영문 혼용 검색은 별도 확인 필요(SPEC 남은 위험과 동일)
