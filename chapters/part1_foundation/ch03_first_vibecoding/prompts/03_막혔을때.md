# 03. 막혔을 때 세 카드

막히면 더 세게 밀어붙이지 말고, 아래 세 카드 중 하나를 고릅니다.

## 카드 ① 되돌리기(revert)

- 무엇: 직전에 동작하던 상태로 되돌립니다.
- 언제 고르나: 방금 수정한 뒤로 더 나빠졌고, 무엇이 바뀌었는지 헷갈릴 때.
- 효과: 깨끗한 출발점에서 다시 시도할 수 있습니다.

## 카드 ② 다시 묻기(re-ask)

- 무엇: 같은 요청을 더 구체적으로 다시 적어 묻습니다. 증상·기대 동작·오류 메시지를 함께 줍니다.
- 언제 고르나: 결과가 방향은 맞는데 어긋날 때, 또는 답이 모호할 때.
- 효과: 맥락이 또렷해져 결과가 좋아집니다.

## 카드 ③ 쪼개기(split)

- 무엇: 한 번에 하려던 일을 더 작은 단계로 나눕니다.
- 언제 고르나: 요청이 커서 어디서 틀렸는지 짚기 어려울 때.
- 효과: 단계마다 확인할 수 있어, 틀린 지점을 빨리 찾습니다.

노트: 세 카드를 다 써도 풀리지 않으면, 막힌 채로 쌓지 말고 잠시 멈춥니다. 부록 C 오류 사전으로 넘어갑니다.
