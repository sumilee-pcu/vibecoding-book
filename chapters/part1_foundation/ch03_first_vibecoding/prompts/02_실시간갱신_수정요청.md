# 02. 실시간 갱신 수정 요청

## 그대로 쓰는 수정 요청

> 지금은 글을 지워도 글자 수가 줄지 않고 그대로 남는다. 입력 칸이 바뀔 때마다 글자 수가 실시간으로 갱신되게 고쳐 줘. 나머지 동작은 그대로 둬.

## 왜 이렇게 요청하는가

- 무엇이 문제인지(증상)를 먼저 말합니다. — "글을 지워도 글자 수가 줄지 않는다"
- 어떻게 되어야 하는지(기대 동작)를 말합니다. — "입력이 바뀔 때마다 실시간 갱신"
- 건드리지 말 것을 못 박습니다. — "나머지 동작은 그대로 둬"

실무 팁: 수정 요청에 "나머지는 그대로 둬"를 붙이면, 고치는 김에 다른 곳까지 바뀌는 일을 줄일 수 있습니다.
