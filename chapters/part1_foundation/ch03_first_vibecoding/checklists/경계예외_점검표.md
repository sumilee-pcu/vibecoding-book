# 경계·예외 점검표

보통 입력만 보면 멀쩡해 보이지만, 경계 경우에서 문제가 드러납니다. 아래를 직접 넣어 봅니다.

## 점검 항목

- ☐ 빈 입력 — 아무것도 적지 않았을 때 0이 나오는가? (오류로 멈추지 않는가)
- ☐ 아주 긴 글 — 긴 문서를 붙여 넣어도 멈추지 않고 동작하는가?
- ☐ 띄어쓰기 — 공백도 글자로 세는가? (의도한 동작인지 확인)
- ☐ 줄바꿈 — 여러 줄로 나뉜 글에서도 수가 맞는가?
- ☐ 지우기 — 글을 지우면 글자 수가 실시간으로 줄어드는가?

## 기록하기

각 항목 옆에 실제로 본 결과를 적어 둡니다. 예: "빈 입력 → 0 정상", "줄바꿈 → 줄바꿈도 1글자로 셈".

노트: 공백·줄바꿈을 세는 것이 맞는지는 도구의 쓰임에 달려 있습니다. 정답을 정하고, 그에 맞게 동작하는지 확인하는 것이 핵심입니다.
