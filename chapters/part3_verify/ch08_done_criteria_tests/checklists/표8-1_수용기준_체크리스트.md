# 표 8-1. 수용 기준 체크리스트

도서 메모 검색 도구를 "끝났다"고 선언하기 전에 항목별로 통과 기준과 남길 증거를 확인합니다.

| 구분 | 점검 항목 | 통과 기준 | 남길 증거 |
| --- | --- | --- | --- |
| 기능 | 제목·저자·메모 검색이 동작하는가 | 세 필드 각각에서 일치 항목을 찾는다 | 정상 3종 테스트 통과 로그 |
| 기능 | 빈 검색어가 전체 목록을 보여 주는가 | 빈 검색어 결과 길이 == 전체 길이 | 경계 테스트 통과 로그 |
| 예외 | 대소문자·공백을 정규화하는가 | python==Python, " python "==python | 경계 2·3 테스트 통과 로그 |
| 예외 | 일치 없을 때 빈 결과를 주는가 | 없는 검색어 결과가 빈 배열 | 실패/없음 테스트 통과 로그 |
| 화면 | 결과 없음 메시지가 보이는가 | 사람이 화면에서 문구 확인 | 화면 캡처 또는 점검 메모 |
| 테스트 | 자동 테스트가 모두 통과하는가 | node --test 전부 pass | 터미널 실행 결과 |
| 되돌리기 | 검색이 원본을 바꾸지 않는가 | 검색 전후 데이터 동일 | 불변 테스트 통과 로그 |
