# 부록 B. 비용 가이드

비용 감각이 있으면 같은 일을 더 싸게 할 수 있습니다.

## 토큰이란

- 토큰(token)은 모델이 글을 처리하는 최소 단위입니다.
- 대략 한글 한두 글자, 또는 영어 한 단어가 한 토큰쯤입니다.
- 주고받는 글이 길수록 토큰이 늘고, 비용도 늘어납니다.

## 요금 구간

| 구간 | 특징 | 어울리는 상황 |
| --- | --- | --- |
| 구독제 | 월정액으로 일정 한도 사용 | 매일 꾸준히 쓰는 경우 |
| 사용량제 | 쓴 만큼 토큰 단위로 과금 | 사용량이 들쭉날쭉한 경우 |
| 무료 체험 구간 | 한도 안에서 무료로 사용 | 처음 익히고 둘러보는 경우 |

## 비용 절감 습관

- 맥락 좁히기 — 필요한 맥락만 줍니다. 불필요하게 긴 입력을 줄입니다. (6장)
- 작업 쪼개기 — 큰 요청을 작은 단계로 나눠, 헛도는 생성을 줄입니다. (5장)
- 자동화 상한 — 자동으로 도는 작업에 상한을 두어 비용이 새지 않게 합니다. (12장)

노트: 최신 금액은 바뀔 수 있으니 공식 요금 안내에서 확인합니다.
